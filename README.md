# Recipes for IIIF Curation Platform

## セットアップ

```
git clone https://github.com/rois-codh/icp-recipe.git
cd icp-recipe
```

その後、ファイルを編集して、masterブランチにpushすると、GitHub Actionsにより自動ビルド＆デプロイが実行されます。

https://rois-codh.github.io/icp-recipe
